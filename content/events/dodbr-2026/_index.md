Day of Data Baton Rouge is a **free, all-day event** hosted by the Baton Rouge User Groups. It offers IT and Microsoft Data Platform professionals, students, and tech enthusiasts the chance to grow their Business Intelligence, Analytics, Azure SQL, and SQL Server skills through expert-led technical training sessions, career development insights, and networking. This community-driven event promotes learning, growth, and connection within the IT community in Baton Rouge.

### Why Attend?

This is a great opportunity to learn from SQL Server Experts, Microsoft MVPs, SQL Server Authors, and Professionals who have been through and can relate to the daily obstacles you face. This one-day training event is unlike any other, providing food, training, and networking.
